CONFERMA PROTOCOLLO
    DATABASE CENTRALE
    Come funziona?
    Leviathan scansiona le profondità del web per trovare Torrent (tramite Debrid) e flussi StreamingCommunity ad alta velocità.
    Cos'è Real-Debrid?
    È il motore turbo. Scarica i file per te sui suoi server veloci per permetterti lo streaming istantaneo senza buffering.
    Cos'è la modalità Solo Database?
    Questa modalità bypassa tutti gli scraper esterni e i motori di ricerca integrati, consultando esclusivamente l'indexer VPS. È la scelta più veloce possibile se cerchi contenuti già presenti nel database di Leviathan.
    Serve il Proxy?
    Non è obbligatorio per l'uso base. Puoi abilitarlo se vuoi usare Debrid Proxy Ghost (per nascondere l'IP a Debrid) o se il tuo ISP blocca specifici servizi.
    StreamingCommunity è incluso?
    Sì, puoi attivarlo dal modulo opzionale. Non richiede più obbligatoriamente il Proxy, ma se hai problemi di blocco geografico potrebbe servirti.
    Cos'è Signal Gate?
    Il Signal Gate limita il numero di risultati mostrati per ogni risoluzione (es. Max 2 risultati 4K, 2 risultati 1080p). Utile per evitare liste infinite di duplicati e avere una selezione pulita.
